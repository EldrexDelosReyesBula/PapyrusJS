# What is Papyr?

Papyr is a beginner-first, production-capable JavaScript application platform that enables developers, students, educators, designers, and startups to build anything from simple websites to large SaaS platforms using a unified, reactive, extensible ecosystem.

### Core Principles
* **Beginner Friendly:** Straightforward, elegant paradigms with zero compile overhead.
* **Browser First:** Operates naturally inside standard browser viewports.
* **Progressive Complexity:** Scales smoothly from classroom mockups to distributed SaaS architectures.
* **Privacy First:** Outfitted with the Web Access Transparency Toolkit (WATT) layer.
* **Long-Term API Stability:** Backed by strict deprecation cycles and stable interfaces.
